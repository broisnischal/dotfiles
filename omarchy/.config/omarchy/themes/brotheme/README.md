# Brotheme

A minimal dark theme for Omarchy. Clean, professional, and easy on the eyes.

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#0f0f0f` | Base background |
| Foreground | `#e0e0e0` | Primary text |
| Black | `#2a2a2a` | Terminal black |
| Red | `#e06c75` | Errors, variables |
| Green | `#98c379` | Strings, success |
| Yellow | `#d19a66` | Warnings, types |
| Blue | `#61afef` | Functions, info |
| Magenta | `#c678dd` | Keywords, storage |
| Cyan | `#56b6c2` | Support, regex |
| White | `#abb2bf` | Secondary text |

## Installation

### Omarchy

To install this theme, use the `omarchy-theme-install` command:

```bash
omarchy-theme-install /home/nees/.config/omarchy/themes/brotheme
```

Or manually symlink the theme:

```bash
ln -s /home/nees/.config/omarchy/themes/brotheme ~/.config/omarchy/themes/omarchy-brotheme
```
